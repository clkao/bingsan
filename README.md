# Bingsan - Name Generator
---
[![build status](https://img.shields.io/travis/clkao/bingsan/master.svg?style=flat-square)](https://travis-ci.org/clkao/bingsan)


## Installation

```
npm install
```

## Running Dev Server

```
npm run dev
```

## Building and Running Production Server

```
npm run build
npm run start
```
